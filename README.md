## 概要
HTMLとJavaScriptで作ったブラックジャックゲームです。

## 遊び方
1. ページを開くと自動でゲームが始まります
2. 「ヒット」でカードを1枚引きます
3. 「スタンド」でCPUのターンに移ります
4. 21に近い方が勝ちです

## ルール
- 21を超えるとバスト（負け）
- CPUは17以上でスタンド

## 使用技術
- HTML
- CSS
- JavaScript

## ファイル構成
- index.html　画面の構造
- style.css　　デザイン
- script.js　　ゲームのロジック

## 工夫・こだわり
- じゃんけんアプリで使った仕組み（ユーザー入力 → ロジック判定 → 結果表示）がそのまま活きるゲームをつくる
- AIを活用（設計確認・骨格コードをリードしてもらいながら、自分で肉付けしていく）
- GitHubを活用してバージョン管理を行う

## 難所・反省
- トランプは、「文字列としての数字」「価値・得点としての数字」「絵柄・記号が示す数字」を読み替える必要がある
- 新しい関数が続出し、「なぜ、そうなるのか」「基本的な仕組み・ロジックは何か」を十分に咀嚼できていない
- とりかかりが遅すぎて、デザイン・アニメーションに必要なcss/javascriptは完全にAI任せになってしまった
- バージョン管理は機能単位で分けると履歴が見やすくなっただろう（時間がなく対応できず）
